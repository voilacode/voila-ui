Alert

An "alert" typically refers to a user interface element that displays a brief message to provide information, feedback, or notifications to the user. It is a simple and commonly used way to communicate with users in a browser environment. Alerts are often used to convey important information or prompt the user for action.

VoilaUI uses different ids to provide the functionality of making the alert appear when clicked on the button which is the id showBtn for the below case

The alert container includes a close button which will come into action by using the id closeBtn in the alert box container

HTML

                        
                            <div class="flex items-center justify-center h-screen ">
                            <button id="showBtn" class="px-4 py-2 text-white duration-300 rounded bg-rose-600 hover:bg-rose-700">Submit</button>
                            </div>
                            <div id="alert" class="fixed bottom-0 right-0 hidden p-4 my-4 border-l-4 rounded-lg shadow-xl ms-4 border-emerald-500 ">
                            <ion-icon name="close" id="closeBtn" class="flex justify-end text-xl ms-auto hover:cursor-pointer" onclick="onToggleMenu(this)"></ion-icon>
                            <h1 class="px-4 text-xl font-semibold text-emerald-600">Success</h1>
                            <p class="p-4 h-fit">You have successfully submitted the details.</p>
                            </div>
